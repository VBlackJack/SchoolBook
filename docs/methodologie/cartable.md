# Faire son cartable

## Pourquoi c'est important ?

!!! info "Les avantages d'un cartable bien préparé"
    - **Moins de stress** le matin
    - **Pas d'oublis** = pas de punitions
    - **Cartable plus léger** (on ne prend que le nécessaire)
    - **Meilleure organisation** = meilleurs résultats

---

## Quand préparer son cartable ?

!!! tip "Règle d'or"
    Préparer son cartable **la veille au soir**, JAMAIS le matin !

### Pourquoi pas le matin ?

| Le matin | La veille |
|----------|-----------|
| Pressé, stressé | Calme, posé |
| On oublie des choses | On peut vérifier |
| On prend trop ou pas assez | On réfléchit |
| Mauvais début de journée | Sérénité |

---

## La méthode en 5 étapes

### Étape 1 : Consulter l'emploi du temps

!!! info "Regarde ton emploi du temps du lendemain"
    Note les matières que tu auras dans l'ordre.

```
Exemple : LUNDI
┌────────────────────────────────────┐
│  8h-9h    : Français               │
│  9h-10h   : Maths                  │
│  10h-11h  : Anglais                │
│  11h-12h  : Histoire-Géo           │
│  14h-15h  : SVT                    │
│  15h-16h  : Techno                 │
└────────────────────────────────────┘
```

### Étape 2 : Consulter l'agenda

!!! warning "Vérifie ton agenda"
    Y a-t-il quelque chose de spécial ?

| À vérifier | Exemple |
|------------|---------|
| **Devoirs à rendre** | Exercices, rédaction |
| **Contrôle** | Feuille, calculatrice |
| **Matériel spécial** | Livre de bibliothèque, équerre |
| **Documents** | Autorisation signée, carnet |

### Étape 3 : Préparer le matériel par matière

| Matière | Matériel type |
|---------|---------------|
| **Français** | Cahier + manuel + Bescherelle |
| **Maths** | Cahier + manuel + calculatrice + kit géométrie |
| **Anglais** | Cahier + workbook + dictionnaire |
| **Histoire-Géo** | Classeur/cahier + manuel |
| **SVT** | Cahier + manuel |
| **Physique-Chimie** | Cahier + manuel + blouse (si TP) |
| **Techno** | Cahier + classeur |
| **Arts plastiques** | Cahier + matériel spécifique |
| **Musique** | Cahier + flûte |
| **EPS** | Tenue de sport (à part) |

### Étape 4 : Vérifier la trousse

!!! tip "Checklist trousse"
    - [ ] Stylos (bleu, noir, rouge, vert)
    - [ ] Crayon à papier
    - [ ] Gomme
    - [ ] Taille-crayon
    - [ ] Règle (15-20 cm)
    - [ ] Surligneurs
    - [ ] Colle
    - [ ] Ciseaux
    - [ ] Correcteur (si autorisé)

### Étape 5 : Vérifier l'agenda et le carnet

!!! danger "Toujours dans le cartable"
    - **Agenda** : pour noter les devoirs
    - **Carnet de correspondance** : obligatoire !

---

## Le matériel permanent

### Ce qui reste toujours dans le cartable

| Élément | Utilité |
|---------|---------|
| **Trousse complète** | Écrire, dessiner, mesurer |
| **Agenda** | Noter les devoirs |
| **Carnet de correspondance** | Communication parents/collège |
| **Petite bouteille d'eau** | S'hydrater |
| **Mouchoirs** | Hygiène |
| **Pochette plastique** | Documents à faire signer |

### Le kit de secours

!!! tip "Dans une petite pochette"
    - Stylo de secours
    - Crayon de secours
    - Quelques feuilles blanches
    - Petit paquet de mouchoirs

## Organisation du cartable

### Comment ranger ?

```
┌─────────────────────────────────────────────┐
│              VUE DU CARTABLE                │
├─────────────────────────────────────────────┤
│  DOS DU CARTABLE (contre le dos)            │
│  ┌─────────────────────────────────────┐    │
│  │ Cahiers/classeurs les + LOURDS      │    │
│  └─────────────────────────────────────┘    │
│                                             │
│  MILIEU                                     │
│  ┌─────────────────────────────────────┐    │
│  │ Cahiers/manuels moyens              │    │
│  └─────────────────────────────────────┘    │
│                                             │
│  AVANT (côté ouverture)                     │
│  ┌─────────────────────────────────────┐    │
│  │ Trousse, agenda, léger              │    │
│  └─────────────────────────────────────┘    │
└─────────────────────────────────────────────┘
```

!!! warning "Important pour le dos"
    Les objets **les plus lourds** doivent être **contre le dos** !

### Ordre des matières

!!! tip "Astuce"
    Range tes affaires dans l'**ordre des cours** :

    - La première matière en dernier (dessus)
    - La dernière matière en premier (dessous)

    Ainsi, tu sors les affaires dans l'ordre !

## Le poids du cartable

### Poids recommandé

!!! danger "Règle de santé"
    Le cartable ne doit pas dépasser **10% du poids de l'élève** !

| Poids de l'élève | Poids max du cartable |
|:----------------:|:---------------------:|
| 30 kg | 3 kg |
| 35 kg | 3,5 kg |
| 40 kg | 4 kg |
| 45 kg | 4,5 kg |

### Comment alléger son cartable

| Problème | Solution |
|----------|----------|
| Trop de livres | Prendre uniquement ceux du jour |
| Classeurs lourds | Utiliser des pochettes par matière |
| Trousse trop pleine | Garder l'essentiel |
| Cahiers + classeurs | Choisir l'un ou l'autre |
| Tout en double | Laisser des affaires au collège (casier) |

### Utiliser son casier (si disponible)

| Au casier | Dans le cartable |
|-----------|------------------|
| Manuels du matin | Manuels de l'après-midi |
| Dictionnaires | Trousse |
| Matériel arts plastiques | Agenda, carnet |
| Affaires d'EPS | Devoirs à rendre |

## Checklist du soir

### Liste à cocher chaque soir

```
☐ J'ai regardé mon emploi du temps de demain
☐ J'ai vérifié mon agenda (devoirs, contrôles)
☐ J'ai préparé les cahiers/classeurs de chaque matière
☐ J'ai les manuels nécessaires
☐ J'ai vérifié ma trousse
☐ J'ai mon agenda et mon carnet de correspondance
☐ J'ai le matériel spécial (si besoin)
☐ J'ai les devoirs à rendre
☐ Mon cartable est rangé (lourd au fond)
```

### Les questions à se poser

| Question | Si oui... |
|----------|-----------|
| Y a-t-il un contrôle ? | Prendre le matériel autorisé |
| Y a-t-il EPS ? | Préparer le sac de sport |
| Y a-t-il un document à faire signer ? | Le mettre en évidence |
| Y a-t-il un livre de bibliothèque à rendre ? | Le mettre dans le cartable |
| Y a-t-il arts plastiques ? | Préparer le matériel demandé |

---

## Les erreurs à éviter

| Erreur | Conséquence | Solution |
|--------|-------------|----------|
| Faire son cartable le matin | Oublis, stress | Préparer la veille |
| Tout prendre "au cas où" | Cartable trop lourd | Prendre uniquement le nécessaire |
| Ne pas vérifier la trousse | Pas de stylo en cours | Vérifier chaque semaine |
| Oublier l'agenda | Pas de devoirs notés | Toujours dans le cartable |
| Ne pas ranger le cartable | Mal de dos | Lourd au fond, léger devant |

## Quiz express

??? question "Quand faut-il préparer son cartable ?"
    **La veille au soir**, jamais le matin !

??? question "Quel pourcentage du poids de l'élève le cartable ne doit-il pas dépasser ?"
    **10%** du poids de l'élève maximum.

??? question "Comment doit-on ranger les objets dans le cartable ?"
    Les objets **les plus lourds contre le dos**, les plus légers devant.

??? question "Que doit-on toujours avoir dans son cartable ?"
    L'**agenda** et le **carnet de correspondance**.

??? question "Quelle est l'erreur la plus courante ?"
    **Faire son cartable le matin** au lieu de la veille au soir.
