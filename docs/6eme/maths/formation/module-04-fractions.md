## Leçon 1 : Comprendre les fractions

### Qu'est-ce qu'une fraction ?

!!! info "Définition"
    Une **fraction** représente une partie d'un tout.

    Elle s'écrit avec deux nombres séparés par une barre :

    $$\frac{numérateur}{dénominateur}$$

    - Le **numérateur** (en haut) = combien de parts on prend
    - Le **dénominateur** (en bas) = en combien de parts égales on a divisé

!!! example "Exemple : 3/4 (trois quarts)"
    ```
    ┌─────────┬─────────┬─────────┬─────────┐
    │█████████│█████████│█████████│         │
    │█████████│█████████│█████████│         │
    └─────────┴─────────┴─────────┴─────────┘
       part 1    part 2    part 3    part 4

    On a divisé en 4 parts égales (dénominateur)
    On a colorié 3 parts (numérateur)
    → On a pris 3/4 du rectangle
    ```

### Le vocabulaire des fractions

| Fraction | Se lit | Signification |
|:--------:|--------|---------------|
| 1/2 | un demi | 1 part sur 2 |
| 1/3 | un tiers | 1 part sur 3 |
| 1/4 | un quart | 1 part sur 4 |
| 3/4 | trois quarts | 3 parts sur 4 |
| 2/5 | deux cinquièmes | 2 parts sur 5 |
| 7/10 | sept dixièmes | 7 parts sur 10 |

!!! tip "Astuce mémo"
    **"Le dénominateur, c'est le chef qui décide en combien on coupe !"**

    **"Le numérateur, c'est le gourmand qui dit combien il en prend !"**

### Fractions égales à 1

!!! warning "Règle importante"
    Quand le numérateur = dénominateur, la fraction vaut **1**

    $$\frac{2}{2} = \frac{3}{3} = \frac{4}{4} = \frac{10}{10} = 1$$

!!! example "Pourquoi ?"
    Si tu coupes une pizza en 4 parts et que tu prends les 4 parts...
    tu as pris toute la pizza = 1 pizza entière !

### Fractions supérieures à 1

!!! info "Fraction > 1"
    Quand le numérateur > dénominateur, la fraction est **supérieure à 1**

!!! example "Exemple : 5/4"
    ```
    Pizza 1 (entière)        Pizza 2 (1 part)
    ┌────┬────┐              ┌────┬────┐
    │████│████│              │████│    │
    ├────┼────┤              ├────┼────┤
    │████│████│              │    │    │
    └────┴────┘              └────┴────┘
      4 parts               + 1 part = 5 parts

    5/4 = 1 pizza entière + 1/4 de pizza
    5/4 = 1 + 1/4
    ```

## Leçon 2 : Fractions et droite graduée

### Placer une fraction sur une droite

!!! tip "Méthode"
    1. Regarde le **dénominateur** : il indique en combien de parts diviser chaque unité
    2. Compte le nombre de parts depuis 0 : c'est le **numérateur**

!!! example "Placer 3/4 sur une droite"
    ```
    Étape 1 : diviser [0;1] en 4 parts égales

    |----|----|----|----|
    0   1/4  2/4  3/4   1

    Étape 2 : compter 3 parts depuis 0

    |----|----|----|----|
    0   1/4  2/4  3/4   1
                   ↑
                  3/4
    ```

!!! example "Placer 5/3 sur une droite"
    ```
    Étape 1 : diviser chaque unité en 3 parts

    |----|----|----|----|----|----|
    0   1/3  2/3   1   4/3  5/3   2

    Étape 2 : compter 5 parts depuis 0

    |----|----|----|----|----|----|
    0   1/3  2/3   1   4/3  5/3   2
                            ↑
                           5/3
    ```

### Fractions égales

!!! info "Fractions équivalentes"
    Des fractions peuvent être **égales** même avec des nombres différents.

    $$\frac{1}{2} = \frac{2}{4} = \frac{3}{6} = \frac{4}{8} = \frac{5}{10}$$

!!! tip "Règle"
    On peut **multiplier** (ou diviser) le numérateur ET le dénominateur
    par le même nombre sans changer la valeur de la fraction.

    $$\frac{1}{2} = \frac{1 \times 2}{2 \times 2} = \frac{2}{4}$$

!!! example "Visualisation"
    ```
    1/2                    2/4
    ┌─────────┬─────────┐  ┌────┬────┬────┬────┐
    │█████████│         │  │████│████│    │    │
    └─────────┴─────────┘  └────┴────┴────┴────┘

    C'est la même quantité colorée !
    ```

## Leçon 3 : Comparer des fractions

### Fractions de même dénominateur

!!! tip "Règle simple"
    Si les dénominateurs sont **égaux**, on compare les **numérateurs**.

!!! example "Compare 3/7 et 5/7"
    Même dénominateur (7), donc on compare 3 et 5.

    3 < 5, donc **3/7 < 5/7**

    ```
    3/7  ┌─┬─┬─┬─┬─┬─┬─┐
         │█│█│█│ │ │ │ │
         └─┴─┴─┴─┴─┴─┴─┘

    5/7  ┌─┬─┬─┬─┬─┬─┬─┐
         │█│█│█│█│█│ │ │
         └─┴─┴─┴─┴─┴─┴─┘
    ```

### Fractions de même numérateur

!!! tip "Règle"
    Si les numérateurs sont **égaux**, la fraction avec le plus **petit dénominateur** est la plus **grande**.

!!! example "Compare 2/3 et 2/5"
    Même numérateur (2).

    3 < 5, donc **2/3 > 2/5**

    ```
    2/3  ┌─────┬─────┬─────┐
         │█████│█████│     │  ← parts plus grandes
         └─────┴─────┴─────┘

    2/5  ┌───┬───┬───┬───┬───┐
         │███│███│   │   │   │  ← parts plus petites
         └───┴───┴───┴───┴───┘
    ```

!!! tip "Astuce mémo"
    **"Plus on partage, moins chacun a !"**

    Si tu partages une pizza en 3, chaque part est plus grosse que si tu la partages en 5.

### Comparer à 1/2

!!! tip "Méthode pratique"
    Pour comparer deux fractions, compare-les d'abord à **1/2** :

    - Si numérateur < moitié du dénominateur → fraction < 1/2
    - Si numérateur > moitié du dénominateur → fraction > 1/2

!!! example "Compare 3/8 et 5/9"
    - 3/8 : moitié de 8 = 4, et 3 < 4, donc 3/8 < 1/2
    - 5/9 : moitié de 9 = 4,5, et 5 > 4,5, donc 5/9 > 1/2

    Donc **3/8 < 5/9**

## Leçon 4 : Fractions décimales

### Définition

!!! info "Fraction décimale"
    Une **fraction décimale** est une fraction dont le dénominateur est
    **10, 100, 1000**, etc.

    Elle peut s'écrire sous forme de nombre décimal.

### Conversion fraction → décimal

| Fraction | Calcul | Décimal |
|:--------:|:------:|:-------:|
| 1/10 | 1 ÷ 10 | **0,1** |
| 7/10 | 7 ÷ 10 | **0,7** |
| 23/100 | 23 ÷ 100 | **0,23** |
| 456/1000 | 456 ÷ 1000 | **0,456** |

!!! tip "Règle rapide"
    - Dénominateur **10** → **1 chiffre** après la virgule
    - Dénominateur **100** → **2 chiffres** après la virgule
    - Dénominateur **1000** → **3 chiffres** après la virgule

### Conversion décimal → fraction

| Décimal | Lecture | Fraction |
|:-------:|---------|:--------:|
| 0,3 | 3 dixièmes | **3/10** |
| 0,47 | 47 centièmes | **47/100** |
| 0,125 | 125 millièmes | **125/1000** |
| 2,5 | 2 unités et 5 dixièmes | **25/10** |

### Fractions usuelles à connaître

!!! warning "À mémoriser !"
    | Fraction | Décimal | Pourcentage |
    |:--------:|:-------:|:-----------:|
    | 1/2 | 0,5 | 50% |
    | 1/4 | 0,25 | 25% |
    | 3/4 | 0,75 | 75% |
    | 1/5 | 0,2 | 20% |
    | 1/10 | 0,1 | 10% |

### Série 1 : Compréhension

1. Quelle fraction du rectangle est colorée ?
   ```
   ┌──┬──┬──┬──┬──┬──┬──┬──┐
   │██│██│██│██│██│  │  │  │
   └──┴──┴──┴──┴──┴──┴──┴──┘
   ```

2. Place 2/3 sur une droite graduée de 0 à 1

3. Trouve une fraction égale à 3/5 avec un dénominateur de 10

??? success "Corrections"
    1. **5/8**
    2. On divise [0;1] en 3 parts égales, 2/3 est à la 2ème graduation
    3. **6/10** (on multiplie par 2 en haut et en bas)

### Série 2 : Comparaisons

1. Range dans l'ordre croissant : 2/5 ; 4/5 ; 1/5 ; 3/5

2. Compare : 3/4 et 3/5

3. Compare : 5/8 et 7/8

??? success "Corrections"
    1. **1/5 < 2/5 < 3/5 < 4/5**
    2. 3/4 **>** 3/5 (même numérateur, 4 < 5)
    3. 5/8 **<** 7/8 (même dénominateur, 5 < 7)

### Série 3 : Fractions décimales

1. Écris 0,75 sous forme de fraction

2. Écris 34/100 sous forme décimale

3. Classe : 1/2 ; 3/10 ; 7/10 ; 0,4

??? success "Corrections"
    1. **75/100** ou **3/4**
    2. **0,34**
    3. 1/2 = 0,5 ; 3/10 = 0,3 ; 7/10 = 0,7 ; 0,4
       → **3/10 < 0,4 < 1/2 < 7/10**

## Bonus : Les fractions égyptiennes

!!! info "Savais-tu ?"
    Les Égyptiens anciens n'utilisaient que des fractions avec **numérateur 1** !

    Ils écrivaient : 2/5 = 1/3 + 1/15

    C'est le **défi** : décomposer une fraction en somme de fractions unitaires.

### Défi

!!! question "Peux-tu écrire ces fractions comme les Égyptiens ?"
    a) 2/3 = 1/? + 1/?

    b) 3/4 = 1/? + 1/?

??? success "Solutions"
    a) 2/3 = **1/2 + 1/6**

    b) 3/4 = **1/2 + 1/4**
