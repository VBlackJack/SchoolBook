## Leçon 1 : Points, droites et segments

### Le vocabulaire de base

!!! info "Les éléments fondamentaux"
    | Élément | Notation | Représentation | Description |
    |---------|:--------:|:--------------:|-------------|
    | Point | A | • | Position dans l'espace |
    | Droite | (AB) ou (d) | ←————→ | Ligne infinie des deux côtés |
    | Demi-droite | [AB) | A————→ | Origine A, infinie d'un côté |
    | Segment | [AB] | A————B | Deux extrémités, longueur finie |

!!! example "Représentation visuelle"
    ```
    Point A          •
                     A

    Droite (AB)      ←——•——•——→
                        A  B

    Demi-droite [AB) •——•——→
                     A  B

    Segment [AB]     •——•
                     A  B
    ```

### Positions relatives de droites

!!! info "Deux droites peuvent être..."
    - **Sécantes** : elles se coupent en un point
    - **Parallèles** : elles ne se coupent jamais (même direction)
    - **Perpendiculaires** : elles se coupent en formant un angle droit

!!! example "Représentation"
    ```
    Sécantes          Parallèles         Perpendiculaires
        \  /                               |
         \/           ————————            _|___
         /\           ————————             |
        /  \                               |
    ```

### Notations

| Relation | Notation | Se lit |
|----------|:--------:|--------|
| A appartient à (d) | A ∈ (d) | "A appartient à d" |
| (d1) parallèle à (d2) | (d1) // (d2) | "d1 parallèle à d2" |
| (d1) perpendiculaire à (d2) | (d1) ⊥ (d2) | "d1 perpendiculaire à d2" |

## Leçon 2 : Les instruments de géométrie

### La règle graduée

!!! tip "Utilisation"
    - Tracer des droites, segments, demi-droites
    - Mesurer des longueurs

!!! warning "Attention"
    - Place le **0** de la règle sur le point de départ
    - Lis la mesure au niveau du point d'arrivée

### L'équerre

!!! tip "Utilisation"
    - Vérifier si un angle est droit
    - Tracer des perpendiculaires

!!! example "Comment tracer une perpendiculaire ?"
    ```
    Étape 1: Place un côté de l'équerre sur la droite
    Étape 2: L'angle droit de l'équerre donne la direction perpendiculaire
    Étape 3: Trace le long de l'autre côté de l'équerre
    ```

### Le compas

!!! tip "Utilisation"
    - Tracer des cercles
    - Reporter des longueurs
    - Construire des figures (triangles, etc.)

!!! example "Tracer un cercle de centre O et rayon 3 cm"
    ```
    Étape 1: Écarte le compas de 3 cm (mesure avec la règle)
    Étape 2: Plante la pointe sur O
    Étape 3: Trace le cercle en faisant tourner le compas
    ```

### Le rapporteur

!!! tip "Utilisation"
    - Mesurer des angles
    - Construire des angles d'une mesure donnée

!!! warning "Attention aux deux graduations !"
    Le rapporteur a deux échelles (0°-180° dans les deux sens).
    Utilise celle qui commence à **0°** sur le côté de l'angle.

## Leçon 3 : Les angles

### Types d'angles

!!! info "Classification des angles"
    | Type | Mesure | Représentation |
    |------|:------:|:--------------:|
    | Angle nul | 0° | Côtés superposés |
    | Angle aigu | Entre 0° et 90° | "Pointu" |
    | Angle droit | 90° | Avec le petit carré |
    | Angle obtus | Entre 90° et 180° | "Large" |
    | Angle plat | 180° | Demi-tour |

!!! example "Représentation visuelle"
    ```
    Aigu (45°)    Droit (90°)    Obtus (120°)    Plat (180°)

       /           |               \
      /           _|___           __\____        ←————————→
     ∠            |               ∠
    ```

!!! tip "Astuce mémo"
    **"Un angle aigu, c'est AIgnu comme une AIguille !"**
    (L'aiguille d'une montre forme des angles aigus quand elle indique 1h, 2h...)

### Mesurer un angle avec le rapporteur

!!! tip "Méthode"
    1. Place le **centre** du rapporteur sur le **sommet** de l'angle
    2. Aligne le **0°** sur un des côtés de l'angle
    3. Lis la mesure là où l'autre côté coupe la graduation

!!! example "Exemple"
    ```
           90°
          __|__
         /  |  \
       /    |    \
      60°   |    120°
     /_____.|.______\
    0°      S      180°

    Si le côté passe par 60° → l'angle mesure 60°
    ```

### Angles particuliers à connaître

| Angle | Mesure | Où le trouve-t-on ? |
|:-----:|:------:|---------------------|
| Droit | 90° | Coin d'une feuille, carré |
| 45° | 45° | Moitié d'un angle droit |
| 60° | 60° | Triangle équilatéral |
| 30° | 30° | Moitié de 60° |

    b) **Obtus** (plus ouvert qu'un angle droit)

    c) **Droit** (environ 90°)

## Leçon 4 : Les figures géométriques

### Les triangles

!!! info "Types de triangles"
    | Type | Propriété | Représentation |
    |------|-----------|:--------------:|
    | Quelconque | Aucune particularité | 3 côtés différents |
    | Isocèle | 2 côtés égaux | 2 côtés marqués = |
    | Équilatéral | 3 côtés égaux | 3 côtés marqués = |
    | Rectangle | 1 angle droit | Petit carré dans l'angle |

!!! example "Représentation"
    ```
    Quelconque      Isocèle       Équilatéral     Rectangle

        △             △               △              △
       /\            /|\             /|\            /|
      /  \          / | \           / | \          / |
     /____\        /__=__\         /_===_\        /__|
    ```

!!! tip "Astuce mémo"
    - **Isocèle** → "**I**so" comme "**I**dentique" (2 côtés identiques)
    - **Équilatéral** → "**Équi**" comme "**Égal**" (tous égaux)

### Les quadrilatères

!!! info "Quadrilatères particuliers"
    | Figure | Propriétés |
    |--------|------------|
    | Carré | 4 côtés égaux + 4 angles droits |
    | Rectangle | Côtés opposés égaux + 4 angles droits |
    | Losange | 4 côtés égaux |
    | Parallélogramme | Côtés opposés parallèles et égaux |
    | Trapèze | 2 côtés parallèles |

!!! example "Représentation"
    ```
    Carré         Rectangle       Losange      Parallélogramme

    ┌───┐         ┌─────┐          ◇            ╱────╲
    │   │         │     │         ╱ ╲          ╱      ╲
    │   │         │     │        ◇   ◇        ╱________╲
    └───┘         └─────┘         ╲ ╱
                                   ◇
    ```

### Le cercle

!!! info "Vocabulaire du cercle"
    | Élément | Définition |
    |---------|------------|
    | Centre | Point au milieu |
    | Rayon | Segment du centre à un point du cercle |
    | Diamètre | Segment passant par le centre (= 2 rayons) |
    | Corde | Segment reliant 2 points du cercle |
    | Arc | Portion du cercle |

!!! example "Représentation"
    ```
           ___________
          /     |     \
         /      |      \
        |   r   |   r   |
        |-------|-------|  ← diamètre (d = 2r)
        |       O       |
         \             /
          \___________/

    O = centre | r = rayon | d = diamètre
    ```

!!! warning "Formule importante"
    **Diamètre = 2 × Rayon**

    d = 2r

## Entraînement

### Série 1 : Vocabulaire

1. Comment s'appelle un segment reliant deux points d'un cercle sans passer par le centre ?
2. Comment note-t-on que deux droites sont parallèles ?
3. Quelle est la mesure d'un angle plat ?

??? success "Corrections"
    1. **Une corde**
    2. **// ou ∥** (exemple : (d1) // (d2))
    3. **180°**

### Série 2 : Angles

1. Classe ces angles : 25°, 95°, 90°, 150°, 180°
2. Un angle mesure 35°. Combien mesure son complément ? (angle + complément = 90°)
3. Un angle mesure 60°. Combien mesure son supplément ? (angle + supplément = 180°)

??? success "Corrections"
    1. Aigus : 25° | Droit : 90° | Obtus : 95°, 150° | Plat : 180°
    2. 90° - 35° = **55°**
    3. 180° - 60° = **120°**

### Série 3 : Figures

1. Combien d'axes de symétrie a un carré ?
2. Tous les angles d'un triangle équilatéral mesurent combien ?
3. Le diamètre d'un cercle est 14 cm. Quel est son rayon ?

??? success "Corrections"
    1. **4 axes** de symétrie
    2. **60°** chacun (180° ÷ 3)
    3. Rayon = 14 ÷ 2 = **7 cm**
