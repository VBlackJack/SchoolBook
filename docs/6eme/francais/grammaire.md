# Grammaire

## Les classes grammaticales

### Les mots variables

Ce sont les mots qui changent de forme (genre, nombre, personne...).

| Classe | Rôle | Exemples |
|--------|------|----------|
| **Nom** | Désigne un être, une chose, une idée | chat, liberté, Paris |
| **Déterminant** | Précède le nom | le, une, mon, ces |
| **Adjectif** | Qualifie le nom | grand, belle, intelligent |
| **Pronom** | Remplace un nom | il, celle-ci, qui |
| **Verbe** | Exprime une action ou un état | manger, être, courir |

### Les mots invariables

Ce sont les mots qui ne changent jamais de forme.

| Classe | Rôle | Exemples |
|--------|------|----------|
| **Adverbe** | Modifie un verbe, un adjectif | vite, très, hier |
| **Préposition** | Introduit un complément | à, de, pour, dans |
| **Conjonction** | Relie des mots ou propositions | et, mais, car, que |
| **Interjection** | Exprime une émotion | Oh ! Hélas ! Bravo ! |

## Le nom

### Nom commun vs nom propre

| Type | Caractéristiques | Exemples |
|------|------------------|----------|
| **Nom commun** | Minuscule, désigne une catégorie | ville, chat, livre |
| **Nom propre** | Majuscule, désigne un être unique | Paris, Médor, Victor Hugo |

### Genre et nombre

!!! note "Le genre"
    - **Masculin** : un livre, le soleil
    - **Féminin** : une table, la lune

!!! note "Le nombre"
    - **Singulier** : un chat (1 seul)
    - **Pluriel** : des chats (plusieurs)

### Formation du pluriel

| Règle | Singulier → Pluriel |
|-------|---------------------|
| Règle générale | chat → chat**s** |
| Mots en -s, -x, -z | souris → souris (invariable) |
| Mots en -au, -eau, -eu | bateau → bateau**x** |
| Mots en -al | cheval → chev**aux** |
| Mots en -ou | clou → clou**s** (sauf bijoux, cailloux, choux, genoux, hiboux, joujoux, poux) |

## Les déterminants

### Les articles

| Type | Masculin sing. | Féminin sing. | Pluriel |
|------|:---:|:---:|:---:|
| **Défini** | le (l') | la (l') | les |
| **Indéfini** | un | une | des |
| **Partitif** | du (de l') | de la (de l') | des |

!!! example "Exemples"
    - **Le** chat dort. (article défini → chat connu)
    - **Un** chat miaule. (article indéfini → chat quelconque)
    - Je mange **du** pain. (article partitif → quantité indéfinie)

### Les déterminants possessifs

| Possesseur | Masc. sing. | Fém. sing. | Pluriel |
|------------|:---:|:---:|:---:|
| je | mon | ma | mes |
| tu | ton | ta | tes |
| il/elle | son | sa | ses |
| nous | notre | notre | nos |
| vous | votre | votre | vos |
| ils/elles | leur | leur | leurs |

### Les déterminants démonstratifs

| Masculin sing. | Féminin sing. | Pluriel |
|:---:|:---:|:---:|
| ce, cet | cette | ces |

!!! tip "Astuce"
    **Cet** devant voyelle ou h muet : cet arbre, cet homme

## L'adjectif qualificatif

### Accord de l'adjectif

!!! warning "Règle fondamentale"
    L'adjectif s'accorde en **genre** et en **nombre** avec le nom qu'il qualifie.

| | Masculin | Féminin |
|---|:---:|:---:|
| **Singulier** | un chat **noir** | une chatte **noire** |
| **Pluriel** | des chats **noirs** | des chattes **noires** |

### Formation du féminin

| Règle | Masculin → Féminin |
|-------|---------------------|
| Règle générale | grand → grand**e** |
| -er → -ère | premier → premiè**re** |
| -eux → -euse | heureux → heure**use** |
| -f → -ve | neuf → neu**ve** |
| Doublement consonne | bon → bon**ne** |

### Place de l'adjectif

!!! note "Généralement après le nom"
    Une voiture **rouge**, un film **intéressant**

!!! note "Certains adjectifs courts avant le nom"
    Un **petit** chat, une **belle** maison, un **vieux** livre

    (beau, bon, grand, gros, jeune, joli, long, mauvais, petit, vieux...)

## La phrase

### Types de phrases

| Type | Fonction | Ponctuation | Exemple |
|------|----------|:---:|---------|
| **Déclarative** | Informer | . | Le chat dort. |
| **Interrogative** | Questionner | ? | Le chat dort-il ? |
| **Exclamative** | Exprimer une émotion | ! | Comme ce chat est mignon ! |
| **Impérative** | Ordonner | . ou ! | Regarde ce chat ! |

### Formes de phrases

| Forme | Exemple |
|-------|---------|
| **Affirmative** | Je mange une pomme. |
| **Négative** | Je **ne** mange **pas** de pomme. |

!!! warning "La négation"
    Elle encadre le verbe : **ne...pas**, **ne...plus**, **ne...jamais**, **ne...rien**

### Phrase simple vs phrase complexe

| Type | Définition | Exemple |
|------|------------|---------|
| **Phrase simple** | 1 seul verbe conjugué | Le chat dort. |
| **Phrase complexe** | Plusieurs verbes conjugués | Le chat dort **et** le chien joue. |

## Les fonctions dans la phrase

### Le sujet

!!! info "Définition"
    Le sujet indique **qui fait l'action** ou **de qui/quoi on parle**.

!!! tip "Pour le trouver"
    Poser la question **"Qui est-ce qui...?"** ou **"Qu'est-ce qui...?"** devant le verbe.

!!! example "Exemple"
    **Le chat** mange sa pâtée. → Qui est-ce qui mange ? → Le chat

### Le verbe

Le verbe est le noyau de la phrase. Il exprime :
- Une **action** : manger, courir, écrire
- Un **état** : être, paraître, sembler, devenir

### Les compléments du verbe

| Fonction | Question | Exemple |
|----------|----------|---------|
| **COD** (Complément d'Objet Direct) | qui ? quoi ? | Je mange **une pomme**. |
| **COI** (Complément d'Objet Indirect) | à qui ? de quoi ? | Je parle **à mon ami**. |
| **Attribut du sujet** | après être, paraître... | Elle est **gentille**. |

!!! tip "Reconnaître le COD"
    Il répond à "qui ?" ou "quoi ?" et n'est pas introduit par une préposition.

### Les compléments de phrase

Ils peuvent être déplacés ou supprimés.

| Type | Question | Exemple |
|------|----------|---------|
| **CC de lieu** | où ? | Je travaille **à la maison**. |
| **CC de temps** | quand ? | Je pars **demain**. |
| **CC de manière** | comment ? | Il court **rapidement**. |
| **CC de cause** | pourquoi ? | Il pleure **de joie**. |

## Quiz express

??? question "Quelle est la classe grammaticale de 'rapidement' ?"
    C'est un **adverbe** (mot invariable qui modifie le verbe).

??? question "Quel est le pluriel de 'cheval' ?"
    **Chevaux** (les mots en -al font -aux au pluriel).

??? question "Dans 'Je mange une pomme', quelle est la fonction de 'une pomme' ?"
    **COD** (Complément d'Objet Direct) → Je mange quoi ? Une pomme.

??? question "'Cette' est un déterminant de quel type ?"
    C'est un **déterminant démonstratif**.

??? question "Transforme en phrase négative : 'Il mange toujours des légumes.'"
    Il **ne** mange **jamais** de légumes.
