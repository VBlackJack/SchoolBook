# Conjugaison

## Les bases de la conjugaison

### Infinitif et groupes

!!! info "L'infinitif"
    C'est la forme "de base" du verbe, non conjuguée : manger, finir, prendre

| Groupe | Terminaison | Exemples |
|:---:|:---:|----------|
| **1er groupe** | -er | chanter, manger, aimer |
| **2ème groupe** | -ir (→ -issons) | finir, choisir, réussir |
| **3ème groupe** | -ir, -oir, -re | partir, voir, prendre |

!!! tip "Reconnaître le 2ème groupe"
    Si "nous **finissons**" (avec -iss-) → 2ème groupe
    Si "nous **partons**" (sans -iss-) → 3ème groupe

### Radical et terminaison

```
    chant  |  ons
    ─────     ────
    radical   terminaison
```

!!! note "Le radical"
    Partie du verbe qui porte le sens (souvent fixe au 1er groupe)

!!! note "La terminaison"
    Partie qui change selon le temps et la personne

## Le présent de l'indicatif

### 1er groupe (-er)

| Personne | CHANTER |
|----------|:---:|
| je | chant**e** |
| tu | chant**es** |
| il/elle/on | chant**e** |
| nous | chant**ons** |
| vous | chant**ez** |
| ils/elles | chant**ent** |

!!! warning "Attention aux verbes en -ger et -cer"
    - manger → nous mang**e**ons (e pour garder le son [ʒ])
    - lancer → nous lan**ç**ons (ç pour garder le son [s])

### 2ème groupe (-ir → -issons)

| Personne | FINIR |
|----------|:---:|
| je | fin**is** |
| tu | fin**is** |
| il/elle/on | fin**it** |
| nous | fin**issons** |
| vous | fin**issez** |
| ils/elles | fin**issent** |

### 3ème groupe

Les terminaisons varient. Voici les plus courantes :

| Personne | PARTIR | PRENDRE | VOULOIR |
|----------|:---:|:---:|:---:|
| je | par**s** | prend**s** | veu**x** |
| tu | par**s** | prend**s** | veu**x** |
| il/elle | par**t** | prend | veu**t** |
| nous | part**ons** | pren**ons** | voul**ons** |
| vous | part**ez** | pren**ez** | voul**ez** |
| ils/elles | part**ent** | prenn**ent** | veul**ent** |

### Verbes ÊTRE et AVOIR

| Personne | ÊTRE | AVOIR |
|----------|:---:|:---:|
| je | suis | ai |
| tu | es | as |
| il/elle/on | est | a |
| nous | sommes | avons |
| vous | êtes | avez |
| ils/elles | sont | ont |

### Autres verbes irréguliers courants

| Personne | ALLER | FAIRE | DIRE | VENIR |
|----------|:---:|:---:|:---:|:---:|
| je | vais | fais | dis | viens |
| tu | vas | fais | dis | viens |
| il/elle | va | fait | dit | vient |
| nous | allons | faisons | disons | venons |
| vous | allez | faites | dites | venez |
| ils/elles | vont | font | disent | viennent |

## L'imparfait de l'indicatif

!!! info "Emploi"
    Action passée qui **dure**, **se répète** ou sert de **décor**.

### Formation

!!! note "Radical"
    = radical de "nous" au présent

!!! note "Terminaisons"
    -ais, -ais, -ait, -ions, -iez, -aient

| Personne | CHANTER | FINIR | ÊTRE |
|----------|:---:|:---:|:---:|
| je | chant**ais** | finiss**ais** | ét**ais** |
| tu | chant**ais** | finiss**ais** | ét**ais** |
| il/elle | chant**ait** | finiss**ait** | ét**ait** |
| nous | chant**ions** | finiss**ions** | ét**ions** |
| vous | chant**iez** | finiss**iez** | ét**iez** |
| ils/elles | chant**aient** | finiss**aient** | ét**aient** |

!!! tip "Astuce"
    Les terminaisons sont les mêmes pour TOUS les verbes !

## Le passé simple

!!! info "Emploi"
    Action passée **ponctuelle**, **brève**, dans un récit.

### 1er groupe

| Personne | CHANTER |
|----------|:---:|
| je | chant**ai** |
| tu | chant**as** |
| il/elle | chant**a** |
| nous | chant**âmes** |
| vous | chant**âtes** |
| ils/elles | chant**èrent** |

### 2ème groupe

| Personne | FINIR |
|----------|:---:|
| je | fin**is** |
| tu | fin**is** |
| il/elle | fin**it** |
| nous | fin**îmes** |
| vous | fin**îtes** |
| ils/elles | fin**irent** |

### Verbes ÊTRE et AVOIR

## Le futur simple

!!! info "Emploi"
    Action qui **se produira** dans l'avenir.

### Formation

!!! note "Règle"
    Infinitif + terminaisons de AVOIR au présent
    (ai, as, a, ons, ez, ont)

| Personne | CHANTER | FINIR | PRENDRE |
|----------|:---:|:---:|:---:|
| je | chanter**ai** | finir**ai** | prendr**ai** |
| tu | chanter**as** | finir**as** | prendr**as** |
| il/elle | chanter**a** | finir**a** | prendr**a** |
| nous | chanter**ons** | finir**ons** | prendr**ons** |
| vous | chanter**ez** | finir**ez** | prendr**ez** |
| ils/elles | chanter**ont** | finir**ont** | prendr**ont** |

### Futurs irréguliers

| Verbe | Radical | Je... |
|-------|:---:|:---:|
| être | ser- | serai |
| avoir | aur- | aurai |
| aller | ir- | irai |
| faire | fer- | ferai |
| venir | viendr- | viendrai |
| voir | verr- | verrai |
| pouvoir | pourr- | pourrai |
| vouloir | voudr- | voudrai |

## Le passé composé

!!! info "Emploi"
    Action passée **terminée**, avec lien avec le présent.

### Formation

!!! note "Règle"
    Auxiliaire (avoir ou être) au présent + **participe passé**

### Avec AVOIR (majorité des verbes)

| Personne | MANGER |
|----------|:---:|
| j' | ai mangé |
| tu | as mangé |
| il/elle | a mangé |
| nous | avons mangé |
| vous | avez mangé |
| ils/elles | ont mangé |

### Avec ÊTRE (verbes de mouvement + pronominaux)

| Personne | PARTIR |
|----------|:---:|
| je | suis parti(e) |
| tu | es parti(e) |
| il/elle | est parti(e) |
| nous | sommes parti(e)s |
| vous | êtes parti(e)(s) |
| ils/elles | sont parti(e)s |

!!! warning "Accord du participe passé"
    - Avec **être** → accord avec le sujet
    - Avec **avoir** → pas d'accord (sauf COD avant)

### Les 14 verbes avec ÊTRE

!!! tip "Moyen mnémotechnique : La maison d'être"
    **D**evenir, **R**evenir, **M**onter, **R**ester, **S**ortir, **V**enir, **A**ller, **N**aître, **D**escendre, **E**ntrer, **R**etourner, **T**omber, **R**entrer, **A**rriver, **M**ourir, **P**artir

    (+ tous les verbes pronominaux : se lever, se laver...)

## L'impératif présent

!!! info "Emploi"
    Donner un **ordre**, un **conseil**, une **interdiction**.

### Formation

3 personnes seulement : tu, nous, vous (sans pronom sujet)

| Personne | CHANTER | FINIR | PRENDRE |
|----------|:---:|:---:|:---:|
| (tu) | chant**e** | fini**s** | prend**s** |
| (nous) | chant**ons** | finiss**ons** | pren**ons** |
| (vous) | chant**ez** | finiss**ez** | pren**ez** |

!!! warning "Pas de -s au 1er groupe (tu)"
    Mange ! Chante ! Écoute ! (mais : Finis ! Prends !)

### Verbes irréguliers

| Verbe | Tu | Nous | Vous |
|-------|:---:|:---:|:---:|
| être | sois | soyons | soyez |
| avoir | aie | ayons | ayez |
| aller | va | allons | allez |
| faire | fais | faisons | faites |

## Tableau récapitulatif des terminaisons

| Temps | 1er groupe | 2ème groupe | 3ème groupe |
|-------|------------|-------------|-------------|
| **Présent** | -e, -es, -e, -ons, -ez, -ent | -is, -is, -it, -issons, -issez, -issent | variable |
| **Imparfait** | -ais, -ais, -ait, -ions, -iez, -aient | idem | idem |
| **Passé simple** | -ai, -as, -a, -âmes, -âtes, -èrent | -is, -is, -it, -îmes, -îtes, -irent | variable |
| **Futur** | -rai, -ras, -ra, -rons, -rez, -ront | idem | idem |

## Quiz express

??? question "Conjugue 'finir' à la 1ère personne du pluriel du présent."
    Nous **finissons**

??? question "Quel est le passé composé de 'elle part' ?"
    Elle **est partie** (verbe de mouvement → être + accord)

??? question "Conjugue 'aller' au futur, 3ème personne du singulier."
    Il/Elle **ira** (futur irrégulier)

??? question "À l'imparfait, comment écrit-on 'nous mangeons' ?"
    Nous **mangions** (radical mang- + terminaison -ions)

??? question "Quel est l'impératif de 'manger' à la 2ème personne du singulier ?"
    **Mange !** (pas de -s au 1er groupe)
