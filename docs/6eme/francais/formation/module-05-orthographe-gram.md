# Module 5 - L'orthographe grammaticale

!!! info "Objectifs du module"
    À la fin de ce module, tu sauras :

    - ✅ Accorder dans le groupe nominal
    - ✅ Accorder le verbe avec le sujet
    - ✅ Distinguer les homophones grammaticaux
    - ✅ Éviter les erreurs les plus fréquentes

    **Durée estimée : 2-3 heures** | **Pré-requis : Modules 1-4**

---

## Leçon 1 : L'accord dans le groupe nominal

### La règle de base

!!! info "Règle"
    Dans le groupe nominal, le **déterminant** et l'**adjectif** s'accordent avec le **nom** en genre et en nombre.

!!! example "Exemples"
    - **un** petit chat → **une** petit**e** chatt**e**
    - **le** grand arbre → **les** grand**s** arbre**s**
    - **cette** belle maison → **ces** belle**s** maison**s**

### Les marques du pluriel

!!! info "Le pluriel des noms"
    | Règle | Singulier | Pluriel |
    |-------|-----------|---------|
    | Cas général | chat | chat**s** |
    | -s, -x, -z | souris, nez | souris, nez |
    | -au, -eau, -eu | bateau, jeu | bateau**x**, jeu**x** |
    | -al | cheval | chev**aux** |
    | -ail (certains) | travail | trav**aux** |

!!! warning "Exceptions à retenir"
    - **-ou** : clou**s**, sauf : bijou**x**, caillou**x**, chou**x**, genou**x**, hibou**x**, joujou**x**, pou**x**
    - **-al** : bal**s**, festival**s**, carnaval**s**, régal**s**

### L'accord de l'adjectif

!!! info "Le féminin des adjectifs"
    | Règle | Masculin | Féminin |
    |-------|----------|---------|
    | Cas général | grand | grand**e** |
    | -e | calme | calme |
    | -er | léger | légèr**e** |
    | -eux | heureux | heureus**e** |
    | -f | sportif | sportiv**e** |
    | Doublement | bon | bon**ne** |

!!! example "Cas particuliers"
    - beau → bel → belle (bel arbre, belle fleur)
    - nouveau → nouvel → nouvelle
    - vieux → vieil → vieille

## Leçon 2 : L'accord sujet-verbe

### La règle de base

!!! info "Règle"
    Le verbe s'accorde **toujours** avec son sujet en personne et en nombre.

!!! example "Exemples"
    - Le chat **dort**. (singulier)
    - Les chats **dorment**. (pluriel)
    - Pierre et Paul **jouent**. (pluriel)

### Les pièges fréquents

!!! warning "Piège 1 : Le sujet éloigné"
    Le sujet peut être éloigné du verbe !

    « Le chien de mes voisins **aboie**. »
    (Le sujet est « le chien », pas « mes voisins »)

!!! warning "Piège 2 : Le sujet inversé"
    Dans les questions ou après certains mots :

    « Où **vont** les enfants ? » (sujet = les enfants)
    « Dans ce jardin **poussent** des fleurs. » (sujet = des fleurs)

!!! warning "Piège 3 : Les sujets coordonnés"
    - Paul **et** Marie **mangent**. (= ils → pluriel)
    - Paul **ou** Marie **viendra**. (= l'un ou l'autre → singulier)

!!! tip "Astuce"
    Encadre le sujet avec « C'est... qui » :

    « C'est **le chien de mes voisins** qui aboie. »

## Leçon 3 : Les homophones grammaticaux

### a / à

!!! tip "Comment les distinguer ?"
    - **a** = verbe avoir → peut être remplacé par « avait »
    - **à** = préposition → ne peut pas être remplacé

!!! example "Exemples"
    - Il **a** faim. → Il *avait* faim. ✅
    - Je vais **à** Paris. → Je vais *avait* Paris. ❌

### et / est

!!! tip "Comment les distinguer ?"
    - **et** = conjonction → peut être remplacé par « et puis »
    - **est** = verbe être → peut être remplacé par « était »

!!! example "Exemples"
    - Marie **et** Paul jouent. → Marie *et puis* Paul jouent. ✅
    - Il **est** gentil. → Il *était* gentil. ✅

### on / ont

!!! tip "Comment les distinguer ?"
    - **on** = pronom (= quelqu'un, nous) → 3ème personne du singulier
    - **ont** = verbe avoir → peut être remplacé par « avaient »

!!! example "Exemples"
    - **On** mange. → *Quelqu'un* mange. ✅
    - Ils **ont** faim. → Ils *avaient* faim. ✅

### son / sont

!!! tip "Comment les distinguer ?"
    - **son** = déterminant possessif → peut être remplacé par « mon »
    - **sont** = verbe être → peut être remplacé par « étaient »

!!! example "Exemples"
    - **Son** livre est beau. → *Mon* livre est beau. ✅
    - Ils **sont** contents. → Ils *étaient* contents. ✅

### ou / où

!!! tip "Comment les distinguer ?"
    - **ou** = choix → peut être remplacé par « ou bien »
    - **où** = lieu / moment → indique un endroit ou un temps

!!! example "Exemples"
    - Tu veux du thé **ou** du café ? → thé *ou bien* café ✅
    - **Où** vas-tu ? → quel endroit ?

### ce / se

!!! tip "Comment les distinguer ?"
    - **ce** = déterminant démonstratif → devant un nom
    - **se** = pronom réfléchi → devant un verbe pronominal

!!! example "Exemples"
    - **Ce** livre est beau. (devant le nom « livre »)
    - Il **se** lave. (verbe pronominal « se laver »)

### ces / ses / c'est / s'est

!!! tip "Comment les distinguer ?"
    | Mot | Nature | Remplacement |
    |-----|--------|--------------|
    | **ces** | dém. pluriel | ce, cette |
    | **ses** | poss. pluriel | son, sa |
    | **c'est** | ce + est | cela est |
    | **s'est** | se + est | s'était |

!!! example "Exemples"
    - **Ces** livres sont à toi. → *Ce* livre est à toi.
    - **Ses** livres sont beaux. → *Son* livre est beau.
    - **C'est** beau ! → *Cela est* beau !
    - Il **s'est** levé. → Il *s'était* levé.

### leur / leurs

!!! tip "Comment les distinguer ?"
    - **leur** (invariable) = pronom COI → remplace « à eux, à elles »
    - **leur/leurs** = déterminant possessif → s'accorde avec le nom

!!! example "Exemples"
    - Je **leur** parle. (pronom, invariable)
    - **Leur** chien est gentil. (possessif, singulier)
    - **Leurs** chiens sont gentils. (possessif, pluriel)

## Exercices guidés - Leçon 3

### Exercice 3.1 : Homophones

!!! question "Complète avec le bon homophone"
    a) Il (a/à) mangé (a/à) midi.
    b) Marie (et/est) gentille (et/est) généreuse.
    c) (On/Ont) dit qu'ils (on/ont) raison.
    d) (Où/Ou) vas-tu, à Paris (où/ou) à Lyon ?

??? success "Correction"
    a) Il **a** mangé **à** midi.
    b) Marie **est** gentille **et** généreuse.
    c) **On** dit qu'ils **ont** raison.
    d) **Où** vas-tu, à Paris **ou** à Lyon ?

### Exercice 3.2 : ces / ses / c'est / s'est

!!! question "Complète"
    a) ___ mon ami qui arrive.
    b) Il a rangé ___ affaires.
    c) ___ livres sont intéressants.
    d) Marie ___ réveillée tôt.

??? success "Correction"
    a) **C'est** mon ami qui arrive.
    b) Il a rangé **ses** affaires.
    c) **Ces** livres sont intéressants.
    d) Marie **s'est** réveillée tôt.

## Entraînement

### Série 1 : Accords

1. Mets au pluriel : un bijou précieux
2. Accorde : « Les feuilles (tomber) en automne. »
3. Accorde : « Le professeur de mes sœurs (être) sévère. »

??? success "Corrections"
    1. **des bijoux précieux**
    2. Les feuilles **tombent** en automne.
    3. Le professeur de mes sœurs **est** sévère.

### Série 2 : Homophones

Complète avec l'homophone correct :

1. Il ___ faim, il n'___ rien mangé. (a/à)
2. ___ frère ___ parti. (son/sont)
3. ___ gâteaux ___ délicieux. (ces/ses/c'est)

??? success "Corrections"
    1. Il **a** faim, il n'**a** rien mangé.
    2. **Son** frère **est** parti.
    3. **Ces** gâteaux **sont** délicieux.

## Tableau récapitulatif des homophones

!!! warning "À mémoriser !"
    | Homophone | Test | Exemple |
    |-----------|------|---------|
    | **a** / à | avait | Il a (avait) faim |
    | **et** / est | et puis / était | Marie et (et puis) Paul |
    | **on** / ont | quelqu'un / avaient | On (quelqu'un) mange |
    | **son** / sont | mon / étaient | Son (mon) livre |
    | **ou** / où | ou bien / endroit | Thé ou (ou bien) café |
    | **ce** / se | devant nom / verbe prono. | Ce livre / Il se lave |

**Q3.** Accorde le verbe : « Les élèves de cette classe (travailler) bien. » (1 pt)

**Q4.** Complète : Il ___ mangé ___ la cantine. (a/à) (2 pts)

**Q5.** Complète : ___ livre ___ passionnant. (ce/se, et/est) (2 pts)

**Q6.** Complète : ___ dit qu'ils ___ tort. (on/ont) (2 pts)

**Q7.** Complète : ___ -tu allé ? (ou/où) (1 pt)

**Q8.** Complète : ___ affaires sont là. (ces/ses) (1 pt)

**Q9.** Complète : ___ mon ami. ___ lui qui m'a aidé. (c'est/s'est) (2 pts)

**Q10.** Complète : Elle ___ endormie. (c'est/s'est) (1 pt)

**Q11-12.** Corrige les erreurs dans ce texte (6 pts) :
« Marie a mangé a midi avec ces amis. Ils ce sont rencontré au restaurant ou ils ont dîné. C'est livres sont sur la table. »

??? success "Corrections"
    Q1. **de beaux chevaux**
    Q2. **des filles sportives**
    Q3. Les élèves de cette classe **travaillent** bien.
    Q4. Il **a** mangé **à** la cantine.
    Q5. **Ce** livre **est** passionnant.
    Q6. **On** dit qu'ils **ont** tort.
    Q7. **Où** es-tu allé ?
    Q8. **Ses** affaires sont là.
    Q9. **C'est** mon ami. **C'est** lui qui m'a aidé.
    Q10. Elle **s'est** endormie.
    Q11-12. Texte corrigé :
    « Marie **a** mangé **à** midi avec **ses** amis. Ils **se** sont rencontr**és** au restaurant **où** ils ont dîné. **Ses** livres sont sur la table. »
