## Leçon 1 : La règle du m devant m, b, p

### La règle

!!! info "Règle"
    Devant **m**, **b**, **p**, on écrit **m** au lieu de **n**.

!!! example "Exemples"
    - e**m**porter (devant p)
    - i**m**possible (devant p)
    - e**m**mener (devant m)
    - a**m**bulance (devant b)
    - no**m**bre (devant b)
    - co**m**prendre (devant p)

### Les exceptions

!!! warning "Exceptions à retenir"
    - bo**n**bon
    - néa**n**moins
    - embo**n**point

!!! tip "Astuce mémo"
    **« Devant m, b, p, n devient m, c'est sympathique ! »**

## Leçon 2 : Les doubles consonnes

### Les consonnes qui doublent souvent

!!! info "Consonnes fréquemment doublées"
    **f, l, m, n, p, r, s, t**

### Règles utiles

!!! tip "Après le son [a]"
    On double souvent après le préfixe **a-** :
    - **a**ccord, **a**pporter, **a**rriver, **a**ttendre

!!! tip "Les mots en -tion"
    - a**tt**ention, addi**t**ion, ques**t**ion

!!! tip "Les mots en -elle, -enne, -erre"
    - nouvelle, ancienne, guerre

!!! tip "Les verbes en -eler et -eter"
    - j'app**elle**, je j**ette** (doublement devant e muet)
    - MAIS : j'ach**è**te, je p**è**le (accent au lieu du doublement)

### Mots courants avec doubles consonnes

!!! warning "À mémoriser"
    | Doubles | Exemples |
    |---------|----------|
    | **cc** | accord, accueil, occasion |
    | **ff** | difficile, différent, affaire |
    | **ll** | belle, appeler, aller |
    | **mm** | grammaire, homme, comment |
    | **nn** | année, bonne, donner |
    | **pp** | apporter, appeler, enveloppe |
    | **rr** | arriver, terrible, erreur |
    | **ss** | classe, possible, laisser |
    | **tt** | lettre, attendre, battre |

## Leçon 3 : Les sons difficiles

### Le son [s]

!!! info "Comment écrire le son [s] ?"
    | Graphie | Position | Exemples |
    |---------|----------|----------|
    | **s** | début de mot ou après consonne | sac, penser |
    | **ss** | entre deux voyelles | classe, poisson |
    | **c** | devant e, i, y | ciel, cygne |
    | **ç** | devant a, o, u | français, leçon |
    | **t** | devant -ion | attention, addition |
    | **sc** | devant e, i | science, scène |

!!! warning "Attention !"
    - Un seul **s** entre deux voyelles fait le son [z] : mai**s**on, poi**s**on
    - Deux **ss** entre deux voyelles font le son [s] : poi**ss**on, cui**ss**on

### Le son [k]

!!! info "Comment écrire le son [k] ?"
    | Graphie | Position | Exemples |
    |---------|----------|----------|
    | **c** | devant a, o, u | car, col, cube |
    | **qu** | devant e, i | que, qui |
    | **k** | mots d'origine étrangère | kilo, kayak |
    | **ch** | mots d'origine grecque | chorale, écho |

### Le son [ʒ] (comme dans "je")

!!! info "Comment écrire le son [ʒ] ?"
    | Graphie | Position | Exemples |
    |---------|----------|----------|
    | **j** | devant a, o, u | jardin, jour |
    | **g** | devant e, i, y | géant, girafe |
    | **ge** | devant a, o, u | orangeade, pigeon |

### Le son [g]

!!! info "Comment écrire le son [g] ?"
    | Graphie | Position | Exemples |
    |---------|----------|----------|
    | **g** | devant a, o, u | gare, gorge |
    | **gu** | devant e, i, y | guitare, guêpe |

## Leçon 4 : Les accents

### Les trois accents

!!! info "Les accents en français"
    | Accent | Nom | Exemple |
    |:------:|-----|---------|
    | **é** | accent aigu | école, été |
    | **è** | accent grave | mère, père |
    | **ê** | accent circonflexe | fête, forêt |

### Quand mettre un accent aigu (é) ?

!!! tip "Règle"
    On met un accent aigu quand le **e** est prononcé [e] fermé (comme dans « été »)
    et qu'il n'est pas suivi d'une consonne double ou d'un x.

!!! example "Exemples"
    - **é**cole, **é**t**é**, caf**é**
    - MAIS : eff**e**t (double f), ex**e**mple (suivi de x)

### Quand mettre un accent grave (è) ?

!!! tip "Règle"
    On met un accent grave quand le **e** est prononcé [ɛ] ouvert (comme dans « mère »)
    devant une syllabe muette.

!!! example "Exemples"
    - m**è**re, p**è**re, fr**è**re
    - j'ach**è**te, je p**è**se

### L'accent circonflexe (ê)

!!! info "Usages"
    - Remplace souvent un ancien **s** : for**ê**t (forest), h**ô**pital (hospital)
    - Distingue des homophones : s**û**r / sur, d**û** / du, m**û**r / mur

### La cédille (ç)

!!! info "Règle"
    La cédille se met sous le **c** devant **a, o, u** pour faire le son [s].

!!! example "Exemples"
    - fran**ç**ais, le**ç**on, re**ç**u
    - MAIS : facile (pas besoin, c devant i)

### Le tréma (ë, ï, ü)

!!! info "Règle"
    Le tréma indique que deux voyelles se prononcent séparément.

!!! example "Exemples"
    - No**ë**l (no-el, pas « noel »)
    - ma**ï**s (ma-is, pas « mais »)
    - capharña**ü**m

## Leçon 5 : Mots souvent mal orthographiés

### Liste des mots difficiles

!!! warning "À mémoriser !"
    | Mot | Attention à... |
    |-----|----------------|
    | **beaucoup** | un seul mot, pas « beau coup » |
    | **toujours** | -ou- et -s final |
    | **longtemps** | g + temps (pas « lontemps ») |
    | **maintenant** | main + tenant |
    | **parmi** | pas de « s » final (≠ hormis) |
    | **malgré** | pas d'accent sur le e |
    | **développer** | un l, deux p |
    | **apercevoir** | un p, un c |
    | **langage** | un seul g |
    | **adresse** | un d, deux s |
    | **apparaître** | deux p, un accent |
    | **enveloppe** | un l, deux p |

### Les familles de mots

!!! tip "Astuce"
    Pour savoir si un mot prend une lettre muette, cherche un mot de la même famille :

    - petit → petit**e** → le t se prononce
    - grand → grand**e** → le d se prononce
    - lait → lait**ier** → le t se prononce
