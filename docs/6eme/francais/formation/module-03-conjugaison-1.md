# Module 3 - La conjugaison (partie 1)

!!! info "Objectifs du module"
    À la fin de ce module, tu sauras :

    - ✅ Identifier les trois groupes de verbes
    - ✅ Conjuguer au présent de l'indicatif
    - ✅ Conjuguer à l'imparfait de l'indicatif
    - ✅ Conjuguer au futur simple de l'indicatif
    - ✅ Connaître les verbes irréguliers courants

    **Durée estimée : 2-3 heures** | **Pré-requis : Module 1**

## Leçon 1 : Les trois groupes de verbes

### Comment les reconnaître ?

!!! info "Les trois groupes"
    | Groupe | Infinitif | Test | Exemples |
    |:------:|-----------|------|----------|
    | **1er** | -ER | (sauf aller) | chanter, manger, aimer |
    | **2ème** | -IR | → nous -issons | finir, choisir, réussir |
    | **3ème** | autres | tout le reste | prendre, venir, être, aller |

### Le test du 2ème groupe

!!! tip "Astuce"
    Un verbe en -IR est du 2ème groupe **seulement si** :
    à la 1ère personne du pluriel, on ajoute **-issons**.

!!! example "Exemples"
    - **finir** → nous fin**issons** ✅ 2ème groupe
    - **venir** → nous ven**ons** ❌ 3ème groupe
    - **partir** → nous part**ons** ❌ 3ème groupe
    - **choisir** → nous chois**issons** ✅ 2ème groupe

### Les verbes du 3ème groupe

!!! warning "Verbes irréguliers à connaître"
    - **Être, avoir, aller, faire**
    - Verbes en -IR (sans -issons) : venir, partir, dormir, sortir...
    - Verbes en -RE : prendre, mettre, dire, écrire, lire...
    - Verbes en -OIR : voir, pouvoir, vouloir, savoir, devoir...

## Leçon 2 : Le présent de l'indicatif

### Les terminaisons

!!! info "Terminaisons du présent"
    | Personne | 1er groupe | 2ème groupe | 3ème groupe |
    |----------|:----------:|:-----------:|:-----------:|
    | je | -e | -is | -s / -x / -e |
    | tu | -es | -is | -s / -x / -es |
    | il/elle | -e | -it | -t / -d / -e |
    | nous | -ons | -issons | -ons |
    | vous | -ez | -issez | -ez |
    | ils/elles | -ent | -issent | -ent |

### 1er groupe : chanter

!!! example "Conjugaison"
    | je chant**e** | nous chant**ons** |
    |--------------|-------------------|
    | tu chant**es** | vous chant**ez** |
    | il chant**e** | ils chant**ent** |

!!! warning "Attention aux particularités !"
    - Verbes en **-ger** : nous mang**e**ons (on garde le e)
    - Verbes en **-cer** : nous commen**ç**ons (cédille)
    - Verbes en **-yer** : j'ennui**e** / j'essa**ie** ou j'essa**ye**

### 2ème groupe : finir

!!! example "Conjugaison"
    | je fin**is** | nous fin**issons** |
    |--------------|-------------------|
    | tu fin**is** | vous fin**issez** |
    | il fin**it** | ils fin**issent** |

### 3ème groupe : verbes courants

!!! warning "Être"
    | je **suis** | nous **sommes** |
    |-------------|-----------------|
    | tu **es** | vous **êtes** |
    | il **est** | ils **sont** |

!!! warning "Avoir"
    | j'**ai** | nous **avons** |
    |----------|----------------|
    | tu **as** | vous **avez** |
    | il **a** | ils **ont** |

!!! warning "Aller"
    | je **vais** | nous **allons** |
    |-------------|-----------------|
    | tu **vas** | vous **allez** |
    | il **va** | ils **vont** |

!!! warning "Faire"
    | je **fais** | nous **faisons** |
    |-------------|------------------|
    | tu **fais** | vous **faites** |
    | il **fait** | ils **font** |

!!! tip "Verbes en -DRE"
    - je pren**ds**, tu pren**ds**, il pren**d**
    - SAUF : verbes en -INDRE et -SOUDRE → je pein**s**, il pein**t**

## Leçon 3 : L'imparfait de l'indicatif

### Les terminaisons

!!! info "Terminaisons de l'imparfait"
    **Toujours les mêmes pour tous les verbes !**

    | Personne | Terminaison |
    |----------|:-----------:|
    | je | -ais |
    | tu | -ais |
    | il/elle | -ait |
    | nous | -ions |
    | vous | -iez |
    | ils/elles | -aient |

!!! tip "Astuce mémo"
    Les terminaisons commencent toutes par **-ai-** sauf nous (-ions) et vous (-iez).

### Formation

!!! tip "Méthode"
    **Radical** = 1ère personne du pluriel au présent (sans -ons) + terminaisons

    - nous chant**ons** → je chant**ais**
    - nous finiss**ons** → je finiss**ais**
    - nous pren**ons** → je pren**ais**

### Exemples

!!! example "1er groupe : chanter"
    | je chant**ais** | nous chant**ions** |
    |-----------------|-------------------|
    | tu chant**ais** | vous chant**iez** |
    | il chant**ait** | ils chant**aient** |

!!! example "2ème groupe : finir"
    | je finiss**ais** | nous finiss**ions** |
    |------------------|---------------------|
    | tu finiss**ais** | vous finiss**iez** |
    | il finiss**ait** | ils finiss**aient** |

!!! warning "Être (exception)"
    | j'**étais** | nous **étions** |
    |-------------|-----------------|
    | tu **étais** | vous **étiez** |
    | il **était** | ils **étaient** |

!!! warning "Attention aux verbes en -ier et -yer"
    - crier → nous cri**i**ons (deux i !)
    - payer → nous pay**i**ons (y + i)

## Leçon 4 : Le futur simple de l'indicatif

### Les terminaisons

!!! info "Terminaisons du futur"
    **Toujours les mêmes pour tous les verbes !**

    | Personne | Terminaison |
    |----------|:-----------:|
    | je | -rai |
    | tu | -ras |
    | il/elle | -ra |
    | nous | -rons |
    | vous | -rez |
    | ils/elles | -ront |

!!! tip "Astuce mémo"
    Les terminaisons ressemblent au verbe **avoir** au présent + **r** :
    ai, as, a, ons, ez, ont → **r**ai, **r**as, **r**a, **r**ons, **r**ez, **r**ont

### Formation

!!! tip "Méthode"
    **Infinitif** + terminaisons

    - chanter → je chanter**ai**
    - finir → je finir**ai**

!!! warning "Verbes en -RE"
    On enlève le **e** final :
    - prendre → je prendr**ai**
    - écrire → j'écrir**ai**

### Exemples réguliers

!!! example "1er groupe : chanter"
    | je chanter**ai** | nous chanter**ons** |
    |------------------|---------------------|
    | tu chanter**as** | vous chanter**ez** |
    | il chanter**a** | ils chanter**ont** |

!!! example "2ème groupe : finir"
    | je finir**ai** | nous finir**ons** |
    |----------------|-------------------|
    | tu finir**as** | vous finir**ez** |
    | il finir**a** | ils finir**ont** |

### Verbes irréguliers au futur

!!! danger "À mémoriser absolument !"
    | Infinitif | Radical futur | Exemple |
    |-----------|:-------------:|---------|
    | **être** | ser- | je **serai** |
    | **avoir** | aur- | j'**aurai** |
    | **aller** | ir- | j'**irai** |
    | **faire** | fer- | je **ferai** |
    | **venir** | viendr- | je **viendrai** |
    | **voir** | verr- | je **verrai** |
    | **pouvoir** | pourr- | je **pourrai** |
    | **vouloir** | voudr- | je **voudrai** |
    | **savoir** | saur- | je **saurai** |
    | **devoir** | devr- | je **devrai** |
    | **courir** | courr- | je **courrai** |
    | **mourir** | mourr- | je **mourrai** |
    | **envoyer** | enverr- | j'**enverrai** |

!!! tip "Astuce pour les doubles R"
    Les verbes **voir, pouvoir, courir, mourir, envoyer** ont un double R au futur.

### Exercice 4.1 : Conjuguer au futur

!!! question "Conjugue au futur"
    a) Demain, je (aller) au cinéma.
    b) Nous (être) en vacances.
    c) Tu (voir) ce film.
    d) Ils (finir) leur travail.

??? success "Correction"

## Entraînement

### Série 1 : Présent

Conjugue au présent :

1. Nous (manger) à midi.
2. Tu (choisir) un livre.
3. Elle (prendre) le métro.
4. Vous (être) prêts.

??? success "Corrections"
    1. Nous **mangeons** à midi.
    2. Tu **choisis** un livre.
    3. Elle **prend** le métro.
    4. Vous **êtes** prêts.

### Série 2 : Imparfait

Conjugue à l'imparfait :

1. Je (regarder) la télévision.
2. Nous (grandir) vite.
3. Ils (avoir) faim.
4. Tu (faire) du sport.

??? success "Corrections"
    1. Je **regardais** la télévision.
    2. Nous **grandissions** vite.
    3. Ils **avaient** faim.
    4. Tu **faisais** du sport.

### Série 3 : Futur

Conjugue au futur :

1. Je (partir) demain.
2. Nous (avoir) le temps.
3. Elle (venir) nous voir.
4. Tu (pouvoir) m'aider.

??? success "Corrections"
    1. Je **partirai** demain.
    2. Nous **aurons** le temps.
    3. Elle **viendra** nous voir.
    4. Tu **pourras** m'aider.
