## Leçon 1 : Les familles de mots

### Définition

!!! info "Famille de mots"
    Une **famille de mots** regroupe tous les mots formés à partir d'un même **radical** (racine).

!!! example "Exemple : la famille de « terre »"
    - **terre** (radical)
    - **terr**ain, **terr**asse, **terr**estre
    - en**terr**er, dé**terr**er, a**terr**ir
    - **terr**ien, **terr**itoire

### Le radical

!!! info "Définition"
    Le **radical** est la partie du mot qui porte le sens principal.
    Il reste (presque) identique dans tous les mots de la famille.

!!! warning "Attention aux variations !"
    Parfois le radical change légèrement :

    - **mer** → **mar**in, **mar**ée, **mar**itime
    - **fleur** → **flor**al, **flor**iste, **fleur**ir
    - **main** → **manu**el, **mani**puler

!!! tip "Utilité"
    Connaître la famille d'un mot aide à :

    - Comprendre des mots nouveaux
    - Trouver l'orthographe (lettres muettes)
    - Enrichir son vocabulaire

## Leçon 2 : Préfixes et suffixes

### Les préfixes

!!! info "Définition"
    Le **préfixe** est un élément placé **au début** du mot.
    Il modifie le sens du radical.

### Préfixes courants

| Préfixe | Sens | Exemples |
|---------|------|----------|
| **in-, im-, il-, ir-** | négation | invisible, impossible, illégal, irrégulier |
| **dé-, dés-** | contraire | défaire, désordre |
| **re-, ré-** | répétition | refaire, réapprendre |
| **pré-** | avant | prévoir, préhistoire |
| **mal-, mé-** | mal | malheureux, méconnaître |
| **sur-** | au-dessus | survoler, surnaturel |
| **sous-** | en-dessous | sous-marin, souligner |
| **para-** | contre | parapluie, parasol |
| **anti-** | contre | antivirus, antibruit |
| **bi-** | deux | bicyclette, bilingue |
| **tri-** | trois | triangle, tricolore |

### Les suffixes

!!! info "Définition"
    Le **suffixe** est un élément placé **à la fin** du mot.
    Il change souvent la nature du mot.

### Suffixes courants

| Suffixe | Forme | Exemples |
|---------|-------|----------|
| **-eur, -euse** | nom de métier | chanteur, coiffeuse |
| **-ier, -ière** | nom de métier | boulanger, jardinière |
| **-iste** | nom de métier | dentiste, fleuriste |
| **-tion, -ation** | nom d'action | création, natation |
| **-ment** | adverbe | lentement, doucement |
| **-able, -ible** | adjectif (possibilité) | mangeable, lisible |
| **-eux, -euse** | adjectif | heureux, joyeuse |

!!! tip "Astuce"
    - Suffixe en **-eur** / **-tion** → souvent un **nom**
    - Suffixe en **-ment** → souvent un **adverbe**
    - Suffixe en **-able** / **-eux** → souvent un **adjectif**

## Leçon 3 : Synonymes et antonymes

### Les synonymes

!!! info "Définition"
    Les **synonymes** sont des mots de sens **proche** ou **identique**.

!!! example "Exemples"
    - maison = demeure = habitation
    - beau = joli = magnifique
    - manger = dévorer = se nourrir

!!! warning "Attention aux nuances !"
    Les synonymes n'ont pas exactement le même sens :

    - **manger** (neutre) ≠ **dévorer** (avec avidité) ≠ **grignoter** (petites quantités)

### Les antonymes

!!! info "Définition"
    Les **antonymes** sont des mots de sens **contraire**.

!!! example "Exemples"
    - grand ≠ petit
    - entrer ≠ sortir
    - jour ≠ nuit
    - possible ≠ impossible

!!! tip "Former un antonyme avec un préfixe"
    - heureux → **mal**heureux
    - possible → **im**possible
    - faire → **dé**faire

## Leçon 4 : Sens propre et sens figuré

### Définitions

!!! info "Sens propre"
    Le **sens propre** est le sens premier, concret, littéral du mot.

!!! info "Sens figuré"
    Le **sens figuré** est un sens imagé, abstrait, dérivé du sens propre.

!!! example "Exemples"
    | Mot | Sens propre | Sens figuré |
    |-----|-------------|-------------|
    | **dévorer** | manger avec avidité | lire avec passion |
    | **tomber** | chuter au sol | tomber amoureux |
    | **clair** | bien éclairé | facile à comprendre |
    | **froid** | basse température | distant, sans émotion |
    | **briller** | émettre de la lumière | exceller, se distinguer |

!!! example "Dans des phrases"
    - **Sens propre** : Le lion **dévore** sa proie.
    - **Sens figuré** : Je **dévore** ce roman passionnant.

### Les expressions idiomatiques

!!! info "Définition"
    Ce sont des expressions au sens figuré qu'on ne peut pas comprendre mot à mot.

!!! example "Exemples"
    - **Avoir le cafard** = être triste (pas un insecte !)
    - **Tomber dans les pommes** = s'évanouir
    - **Poser un lapin** = ne pas venir à un rendez-vous
    - **Avoir la tête dans les nuages** = être distrait
    - **Casser les pieds** = ennuyer

## Exercices guidés - Leçon 4

### Exercice 4.1 : Sens propre ou figuré ?

!!! question "Indique si le mot est au sens propre (P) ou figuré (F)"
    a) La lumière du soleil est aveuglante. (aveuglante)
    b) Sa beauté est aveuglante. (aveuglante)
    c) Il a un cœur de pierre. (pierre)
    d) Cette pierre est lourde. (pierre)

??? success "Correction"
    a) **P** - la lumière éblouit vraiment les yeux
    b) **F** - signifie « extraordinaire, impressionnante »
    c) **F** - signifie « insensible, sans pitié »
    d) **P** - une vraie pierre, objet concret

## Leçon 5 : Les niveaux de langue

### Les trois niveaux

!!! info "Registres de langue"
    | Niveau | Usage | Exemple |
    |--------|-------|---------|
    | **Familier** | entre amis, oral | une bagnole, bouffer |
    | **Courant** | quotidien | une voiture, manger |
    | **Soutenu** | écrit, officiel | un véhicule, se restaurer |

### Exemples comparés

| Familier | Courant | Soutenu |
|----------|---------|---------|
| bagnole | voiture | véhicule |
| bouffer | manger | se restaurer |
| fric | argent | pécule |
| piaule | chambre | appartement |
| bosser | travailler | œuvrer |
| rigoler | rire | s'esclaffer |
| se barrer | partir | se retirer |
| moche | laid | disgracieux |

!!! tip "Quand utiliser quel niveau ?"
    - **Familier** : entre amis, dans les dialogues de récits
    - **Courant** : à l'école, dans la vie quotidienne
    - **Soutenu** : dans les rédactions, les lettres officielles

1. Trouve 3 mots de la famille de « fleur ».
2. Quel est le radical commun : nageur, nager, natation ?
3. Trouve un mot de la famille de « terre » avec un préfixe.

??? success "Corrections"
    1. fleuriste, floral, fleurir, effleurer...
    2. Radical : **nag-** / **nat-**
    3. atterrir, enterrer, déterrer...

### Série 2 : Préfixes et suffixes

1. Quel est le contraire de « heureux » avec un préfixe ?
2. Forme un adverbe à partir de « doux ».
3. Décompose : « imprévisible ».

??? success "Corrections"
    1. **mal**heureux
    2. douce**ment**
    3. **im** (préfixe) + **prév** (radical de prévoir) + **isible** (suffixe)

### Série 3 : Sens propre et figuré

1. « Il a le cœur sur la main » est au sens propre ou figuré ?

??? success "Corrections"
    1. **Sens figuré** (= il est généreux)
    2. Brûler d'envie = désirer intensément
    3. Être distrait, rêveur, ne pas être attentif
