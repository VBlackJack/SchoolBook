# Orthographe

## Les accords

### Accord sujet-verbe

!!! warning "Règle fondamentale"
    Le verbe s'accorde toujours avec son **sujet** en personne et en nombre.

| Sujet | Verbe |
|-------|-------|
| Je | chant**e** |
| Tu | chant**es** |
| Il/Elle | chant**e** |
| Nous | chant**ons** |
| Vous | chant**ez** |
| Ils/Elles | chant**ent** |

!!! tip "Attention aux pièges"
    - **Sujet inversé** : Dans le ciel **volent** les oiseaux.
    - **Sujet éloigné** : Le chat de mes voisins **miaule**.
    - **Sujet composé** : Pierre et Marie **jouent**.

### Accord de l'adjectif

!!! note "Règle"
    L'adjectif s'accorde en **genre** (masculin/féminin) et en **nombre** (singulier/pluriel) avec le nom.

| | Singulier | Pluriel |
|---|---|---|
| **Masculin** | un pull **vert** | des pulls **verts** |
| **Féminin** | une robe **verte** | des robes **vertes** |

!!! warning "Avec plusieurs noms"
    - Masc. + Masc. = Masc. pluriel : un chat et un chien **noirs**
    - Fém. + Fém. = Fém. pluriel : une robe et une jupe **vertes**
    - Masc. + Fém. = Masc. pluriel : un chat et une chatte **noirs**

### Accord du participe passé

| Auxiliaire | Règle | Exemple |
|------------|-------|---------|
| **ÊTRE** | Accord avec le sujet | Elle est **partie**. |
| **AVOIR** | Pas d'accord (sauf COD avant) | Elle a **mangé** une pomme. |

!!! example "Avec avoir + COD avant"
    La pomme que j'ai **mangée**. (COD "que" = la pomme, féminin)

## Les homophones grammaticaux

### a / à

| Mot | Nature | Astuce | Exemple |
|:---:|--------|--------|---------|
| **a** | verbe avoir | Remplacer par "avait" | Il **a** faim. (Il avait faim ✓) |
| **à** | préposition | Ne peut pas être remplacé | Je vais **à** Paris. |

### et / est

| Mot | Nature | Astuce | Exemple |
|:---:|--------|--------|---------|
| **et** | conjonction | Remplacer par "et puis" | Pierre **et** Marie. |
| **est** | verbe être | Remplacer par "était" | Il **est** grand. (Il était grand ✓) |

### son / sont

| Mot | Nature | Astuce | Exemple |
|:---:|--------|--------|---------|
| **son** | déterminant possessif | Remplacer par "mon/ton" | **Son** chat dort. |
| **sont** | verbe être | Remplacer par "étaient" | Ils **sont** là. |

### on / ont

| Mot | Nature | Astuce | Exemple |
|:---:|--------|--------|---------|
| **on** | pronom (= nous, quelqu'un) | Remplacer par "il" | **On** mange. (Il mange ✓) |
| **ont** | verbe avoir | Remplacer par "avaient" | Ils **ont** faim. |

### ou / où

| Mot | Nature | Astuce | Exemple |
|:---:|--------|--------|---------|
| **ou** | conjonction (choix) | Remplacer par "ou bien" | Thé **ou** café ? |
| **où** | adverbe (lieu/temps) | Indique un endroit | **Où** vas-tu ? |

### ce / se

| Mot | Nature | Astuce | Exemple |
|:---:|--------|--------|---------|
| **ce** | déterminant/pronom | Devant un nom ou "que/qui" | **Ce** livre est beau. |
| **se** | pronom réfléchi | Devant un verbe pronominal | Il **se** lave. |

### ces / ses / c'est / s'est

| Mot | Nature | Exemple |
|:---:|--------|---------|
| **ces** | dém. pluriel (ce/cette) | **Ces** livres sont beaux. |
| **ses** | poss. pluriel (son/sa) | **Ses** livres sont rangés. |
| **c'est** | ce + est | **C'est** beau ! |
| **s'est** | se + est (verbe pronom.) | Il **s'est** levé. |

### leur / leurs

| Mot | Nature | Exemple |
|:---:|--------|---------|
| **leur** (invariable) | pronom COI | Je **leur** parle. |
| **leur/leurs** | déterminant possessif | **Leur** chat / **Leurs** chats |

## Le pluriel des noms

### Règle générale

!!! note "Ajouter un -s"
    livre → livre**s**, chat → chat**s**

### Cas particuliers

| Terminaison | Pluriel | Exemples |
|-------------|---------|----------|
| **-s, -x, -z** | Invariable | souris → souris, nez → nez |
| **-au, -eau, -eu** | + x | bateau → bateau**x**, jeu → jeu**x** |
| **-al** | -aux | cheval → chev**aux**, journal → journ**aux** |
| **-ail** | -ails | détail → détail**s** |
| **-ail** (exceptions) | -aux | travail → trav**aux**, vitrail → vitr**aux** |
| **-ou** | + s | clou → clou**s** |

!!! warning "Les 7 exceptions en -oux"
    **B**ijoux, **c**ailloux, **c**houx, **g**enoux, **h**iboux, **j**oujoux, **p**oux

    Moyen mnémotechnique : "Viens mon **chou**, mon **bijou**, sur mes **genoux** avec tes **joujoux**, et ne jette pas de **cailloux** sur ce **hibou** plein de **poux** !"

## Le féminin des noms et adjectifs

### Règle générale

!!! note "Ajouter un -e"
    ami → ami**e**, grand → grand**e**

### Cas particuliers

| Terminaison masc. | Terminaison fém. | Exemples |
|-------------------|------------------|----------|
| **-er** | -ère | boulanger → boulangèr**e** |
| **-eur** | -euse | chanteur → chante**use** |
| **-teur** | -trice | acteur → actr**ice** |
| **-eux** | -euse | heureux → heure**use** |
| **-f** | -ve | neuf → neu**ve** |
| **-c** | -che/-que | blanc → blan**che** |
| consonne doublée | | bon → bon**ne**, gros → gros**se** |

### Féminins irréguliers

| Masculin | Féminin |
|:---:|:---:|
| beau | belle |
| nouveau | nouvelle |
| vieux | vieille |
| fou | folle |
| frais | fraîche |

## Les lettres muettes

### Le -e muet final

!!! tip "Quand mettre un -e final ?"
    - Féminin des adjectifs : grand → grand**e**
    - Certains noms : la vie, la joie, une amie

### Les consonnes muettes finales

!!! tip "Trouver la consonne muette"
    Chercher un mot de la même famille !

| Mot | Famille | Lettre muette |
|-----|---------|:---:|
| gran**d** | grandeur | d |
| lon**g** | longueur | g |
| tem**ps** | temporel | ps |
| respe**ct** | respecter | ct |
| boi**s** | boisé | s |

## Les accents

### Les trois accents

| Accent | Nom | Exemples |
|:---:|------|----------|
| **é** | accent aigu | école, été, café |
| **è** | accent grave | père, mère, fièvre |
| **ê** | accent circonflexe | fête, forêt, être |

### Règles de base

!!! note "Accent aigu (é)"
    Son [e] fermé, souvent en début ou fin de mot : **é**cole, caf**é**

!!! note "Accent grave (è)"
    Son [ɛ] ouvert, devant une syllabe muette : p**è**re, fi**è**vre

!!! note "Accent circonflexe (ê)"
    Souvent trace d'un ancien "s" : for**ê**t (forest), h**ô**pital (hospital)

## La ponctuation

| Signe | Nom | Usage |
|:---:|------|-------|
| **.** | Point | Fin de phrase déclarative |
| **?** | Point d'interrogation | Fin de phrase interrogative |
| **!** | Point d'exclamation | Fin de phrase exclamative ou impérative |
| **,** | Virgule | Sépare des éléments, pause courte |
| **;** | Point-virgule | Sépare des propositions liées |
| **:** | Deux-points | Introduit une explication, une liste |
| **« »** | Guillemets | Encadrent les paroles |
| **-** | Tiret | Change de locuteur dans un dialogue |
| **( )** | Parenthèses | Information supplémentaire |

## Quiz express

??? question "Complète : Il ... parti hier. (a/à)"
    Il **est** parti hier. (Attention, c'est le verbe être !)

??? question "Complète : Elle ... ses devoirs. (a/à)"
    Elle **a** ses devoirs. (Remplaçable par "avait")

??? question "Quel est le pluriel de 'bijou' ?"
    **Bijoux** (exception en -oux)

??? question "Complète : ... livres sont intéressants. (ces/ses)"
    Les deux sont possibles !
    - **Ces** livres = ceux-là (démonstratif)
    - **Ses** livres = les siens (possessif)

??? question "Accord : Les filles sont (parti) tôt."
    Les filles sont **parties** (auxiliaire être → accord avec le sujet féminin pluriel)
